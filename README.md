# 🔐 Wazuh SIEM com Rocky Linux

Implementação prática do **Wazuh SIEM** para monitoramento de segurança em ambiente Linux.

---

## 📌 Objetivo

Criar um ambiente de segurança capaz de:

* Monitorar eventos do sistema
* Detectar atividades suspeitas
* Gerar alertas em tempo real

---

## 🏗️ Estrutura

* 📂 docs/instalacao.md → Instalação do ambiente
* 📂 docs/configuracao.md → Configuração do Wazuh
* 📂 docs/testes.md → Simulação de ataques/eventos
* 📂 docs/troubleshooting.md → Problemas e soluções

---

## 🖥️ Ambiente

* VirtualBox
* Rocky Linux
* Wazuh Server + Dashboard
* Acesso via PuTTY

---

## 🚀 Tecnologias

* Wazuh (SIEM)
* Linux
* Monitoramento de logs

---

## 📊 Resultado

* Dashboard funcional
* Agentes conectados
* Eventos sendo capturados
